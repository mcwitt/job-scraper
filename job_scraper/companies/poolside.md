# Poolside

## Overview

AI company building foundation models purpose-built for software
engineering. Focused on enterprise B2B, selling AI coding automation
to large organizations with 5,000+ developers, including banks and
defense contractors. Founded in 2023 by Jason Warner (former GitHub
CTO) and Eiso Kant.

## Technical Focus

- Code-specialized foundation models (Malibu for complex tasks,
  Point for sub-200ms IDE completion)
- Reinforcement Learning from Code Execution Feedback (RLCEF)
- Enterprise-deployed models with private code ingestion
- License-safe training (excludes GPL/AGPL code)
- Multi-file code generation, test creation, and refactoring

## Scale & Stage

Private, Series B+. Raised $500M Series B in October 2024 at $3B
valuation (Bain Capital). Raising ~$2B at $12B valuation as of
October 2025, with up to $1B from Nvidia. ~303 employees. Hit
$50M revenue in March 2025.

## Hiring

Research and engineering organization building code-focused

code understanding, program synthesis, and large-scale model
training.

Enterprise go-to-market means also hiring for sales, solutions
engineering, and customer success roles targeting large developer
organizations.
