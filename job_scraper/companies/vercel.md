# Vercel

## Overview

Frontend cloud platform for building, deploying, and scaling
web applications. Creator and maintainer of Next.js, the most
popular React framework. Founded in 2015 by Guillermo Rauch
(also created Socket.IO and Mongoose). Headquartered in San
Francisco, California. Remote-friendly with a distributed
global workforce.

## Technical Focus

- Frontend cloud: build, preview, and ship web applications
- Next.js framework (React, server components, edge runtime)
- v0 — AI-powered UI generation from text prompts using
  React, Tailwind CSS, and shadcn/ui
- Vercel AI SDK — open-source TypeScript toolkit for building
  AI-powered applications and agents
- Edge network and CDN with global PoPs
- Serverless and edge functions (Node.js, Go, Python, Ruby)
- Incremental Static Regeneration and image optimization
- Infrastructure: DDoS protection, Web Application Firewall,
  bot management (BotID with Kasada)

## Scale & Stage

Private. Series F ($300M, September 2025) at $9.3B valuation,
co-led by Accel and GIC. Total funding ~$863M across 6 rounds.
~870 employees. Revenue ~$200M in 2025, up from ~$100M in
2024 — roughly 100% YoY growth.

## Hiring

Engineering-heavy org spanning infrastructure, CDN/edge,
fullstack, IAM/security, and AI product teams. Remote roles
available globally. Strong open-source culture — contributions
to Next.js, Turborepo, SWR, and the AI SDK are valued. High
bar for web performance and developer experience expertise.
