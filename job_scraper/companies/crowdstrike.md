# CrowdStrike

## Overview

Cybersecurity company providing cloud-native endpoint
protection, threat intelligence, and incident response.
Built around the AI-native Falcon platform, which unifies
endpoint, identity, cloud, and data protection into a
single agent. Founded 2011 by George Kurtz, Dmitri
Alperovitch, and Gregg Marston. HQ in Austin, Texas.

## Technical Focus

- AI-native Falcon platform (single lightweight agent)
- Endpoint detection and response (EDR/XDR)
- Identity security (Falcon Next-Gen Identity, SGNL
  acquisition)
- Cloud security and workload protection
- Threat intelligence and adversary tracking
- Agentic AI for security (Charlotte AI, Threat AI)
- Data protection and security analytics at scale

## Scale & Stage

Public (NASDAQ: CRWD). ~10,000+ employees across 170+
countries. FY2026 revenue $4.81B, up ~22% YoY. Ending
ARR reached $5.25B -- fastest pure-play cybersecurity
company to that milestone. Net new ARR exceeded $1B for
the first time. Market cap ~$109B (March 2026).

## Hiring

Remote-first company with a high-trust, autonomous
culture. Engineering roles span distributed systems,
cloud infrastructure, data engineering, detection
engineering, and site reliability. Emphasis on solving
customer problems end-to-end rather than siloed work.
Core values: customer obsession, relentless innovation.

## Recent Context

Acquired identity security startup SGNL for ~$740M
(January 2026). Partnered with Nord Security for SMB
cybersecurity. Named inaugural partner in Google's
Unified Security Partner Program. Launched Threat AI,
an agentic threat intelligence system. Running a
cybersecurity startup accelerator with AWS and NVIDIA.
