# Elastic

## Overview

Search AI company behind Elasticsearch, Kibana, and the
ELK stack. Provides search, observability, and security
solutions on the Elastic Search AI Platform. Founded 2012
in Amsterdam by Shay Banon, Uri Boness, Simon Willnauer,
and Steven Schuurman. HQ in Amsterdam and San Francisco.
Used by more than 50% of the Fortune 500.

## Technical Focus

- Elasticsearch distributed search and analytics engine
- Kibana visualization and dashboarding
- Observability (logs, metrics, APM, synthetic monitoring)
- Security analytics and SIEM/SOAR
- AI-powered search: vector search, semantic search,
  retrieval-augmented generation (RAG)
- GPU-accelerated vector database (Elasticsearch GPU plugin)
- Agent Builder context engineering platform for AI agents
- Elastic Cloud (managed SaaS on AWS, GCP, Azure)

## Scale & Stage

Public (NYSE: ESTC). ~4,890 employees as of Jan 2026.
FY2026 revenue guided ~$1.71-1.72B; Q3 FY2026 revenue
$450M (+18% YoY). Market cap ~$5.5B. Subscription revenue
makes up ~95% of total. Free cash flow positive.

## Hiring

Fully distributed company with no mandatory office.
Engineering roles span search, observability, security,
cloud infrastructure, and ML/AI. Known for strong
open-source roots and a collaborative culture. Glassdoor
rating ~4.1 for work-life balance. Hiring process
typically includes recruiter screen, technical interviews,
and a team-fit conversation.

## Recent Context

Launched Agent Builder (technical preview) for building
AI agents using enterprise data. Released Elasticsearch
GPU plugin for 12x faster vector indexing. Introduced
Elastic Inference Service for self-managed clusters.
Deepened partnership with Siren for AI-driven national
security investigations. Partnered with Dell and NVIDIA
on an integrated AI data platform stack.
