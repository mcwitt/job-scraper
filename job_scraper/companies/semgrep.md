# Semgrep

## Overview

Developer security company building an application security platform
centered on static analysis (SAST), software composition analysis
(SCA), and secrets detection. The core engine is open source and uses
a pattern-matching approach where rules look like the source code they
match, supporting 30+ languages. The commercial platform adds
cross-file data-flow analysis, AI-powered triage, and managed scans.

## Technical Focus

- Static analysis with cross-file, cross-function taint tracking
- AI-assisted triage and remediation of security findings
- Software composition analysis and secrets detection
- Developer-first security tooling integrated into CI/CD and editors
- Rule authoring DSL that resembles source code patterns
