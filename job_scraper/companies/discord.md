# Discord

## Overview

Real-time communication platform for text, voice, and video,
used by 200M+ monthly active users across gaming, education,
and community spaces. Founded in 2015 by Jason Citron and
Stanislav Vishnevskiy. Headquartered in San Francisco, CA.
Humam Sakhnini (ex-Activision Blizzard) became CEO in April
2025, with Citron moving to a board/advisor role.

## Technical Focus

- Real-time messaging gateway built on Elixir and Rust,
  handling millions of concurrent connections
- Python-based API layer and backend services
- Audio/video infrastructure using Elixir control plane
  with C++ media streaming across 1,000+ nodes
- Rust NIFs via Rustler for performance-critical paths
  within the BEAM VM
- Social SDK for native game integration (account linking,
  presence, rich activity feeds)
- In-app commerce platform for digital goods and gifting
- Cloud streaming for Instant Play Quests (try-before-buy)

## Scale & Stage

Private; confidentially filed for IPO in January 2026 with
Goldman Sachs and JPMorgan as lead underwriters. Raised
~$1B across 12 rounds; last valued at $15B (Series I, 2021).
Current estimated valuation $7-10B. ~$725M ARR at end of
2024, up ~21% YoY. Approximately 900-1,000 employees after
a 17% workforce reduction in early 2024.

## Hiring

Engineering roles span real-time systems, infrastructure,
consumer product, and revenue/growth. Interview process
includes recruiter screen, technical coding round (often
chat-service focused), and system design. Values product
sense, user empathy, and ownership.
