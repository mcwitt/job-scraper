# Snyk

## Overview

Developer security platform covering static analysis (SAST), software
composition analysis (SCA), container security, infrastructure as code
scanning, and dynamic application security testing (DAST). Founded in
2015 in Tel Aviv and London, now headquartered in Boston. Positioned
as a developer-first security tool that integrates into IDEs, CLIs,
and CI/CD pipelines.

## Technical Focus

- AI-native SAST (Snyk Code) with real-time scanning
- Software composition analysis for open source dependencies
- Container and infrastructure-as-code security scanning
- Dynamic application security testing (Snyk API & Web)
- Integration with AI coding assistants for secure code generation
- Agentic AI security governance tools
