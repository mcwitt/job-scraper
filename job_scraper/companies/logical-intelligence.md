# Logical Intelligence

## Overview

AI research company building language-free,
mathematically grounded models for formal code
verification. Founded by Eve Bodnia (PhD quantum
information, UC Santa Barbara). HQ in San Francisco.

## Technical Focus

- Energy-Based Models (EBMs) that reason in structured
  state space instead of token prediction
- Aleph — AI agent that converts code into
  mathematical proofs verified to 100% accuracy
- Noa — audit agent for finding bugs in existing code
- 76% on Putnam Benchmark for mathematical reasoning
- Non-autoregressive, non-LLM architecture

## Scale & Stage

- Private; seeking funding at $1B+ valuation
- Investors: Cathay Innovation, Pebblebed
- Yann LeCun (Turing Award) chairs Technical Research
  Board; team includes a Fields Medal winner and
  programming world champion
- Small team, rapidly expanding
