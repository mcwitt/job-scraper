# Modal

## Overview

Serverless cloud compute platform purpose-built for AI and data workloads.
Developers define infrastructure as Python code, and Modal handles container
orchestration, GPU scheduling, and scaling without managing servers or VMs.
Founded by Erik Bernhardsson (former Spotify ML lead and Better.com CTO).

## Technical Focus

- Serverless container orchestration with GPU scheduling
- Python-native infrastructure-as-code (decorators, not YAML)
- Optimized inference runtime with low cold-start latency
- On-demand GPU provisioning and autoscaling
