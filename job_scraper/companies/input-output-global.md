# Input Output Global

## Overview

Blockchain research and engineering company best
known as the developer of the Cardano blockchain.
Founded in 2015 by Charles Hoskinson (Ethereum
co-founder). Originally Input Output Hong Kong
(IOHK), rebranded to IOG in 2021 when it moved
headquarters from Hong Kong to Wyoming, USA.

## Technical Focus

- Cardano blockchain (Haskell-based node, Ouroboros
  proof-of-stake consensus protocol)
- Plutus smart contract platform (Haskell/GHC-based)
- Formal methods and peer-reviewed research
- Ouroboros Leios scalability protocol
- Midnight privacy-focused sidechain

## Scale & Stage

Private, self-funded through Cardano treasury and
ecosystem revenue. No traditional VC funding.
Approximately 200-500 employees, fully remote
and globally distributed. Strong academic research
arm with published papers and university
partnerships (e.g., Tokyo Tech, Edinburgh).

## Hiring

Research-oriented engineering culture. Hires
Haskell developers, formal methods researchers,
distributed systems engineers, and cryptographers.
Remote-first with globally distributed teams.
Unusual in requiring functional programming
expertise (Haskell) for core engineering roles.

## Recent Context

Advancing Ouroboros Leios for next-generation
scalability with Haskell and Rust simulations.
Building ledger-on-disk to reduce Cardano node
RAM requirements. Pursuing node diversity via
the Acropolis modular node project. Developing
smart contract verification tooling for Plutus.
