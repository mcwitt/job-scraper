# Linear

## Overview

Issue tracking and project management tool designed for
modern software teams. Founded in 2019 by Karri Saarinen
(CEO), Tuomas Artman (CTO), and Jori Lallo. Fully
distributed company, headquartered in San Francisco.
Known for speed, polish, and a developer-first approach
to product development workflows.

## Technical Focus

- Real-time sync engine with proprietary websocket
  framework built for speed and offline support
- React-based UI with virtualized rendering, drag & drop,
  and keyboard-first interaction design
- AI integrations including MCP server support for coding
  agents (Cursor, Claude) and auto-triage
- Advanced search with combinatorial AND/OR filtering
- Integrations with GitHub, GitLab, Slack, Figma, Sentry

## Scale & Stage

Private. Series C ($82M) in June 2025, $134M total raised
at a $1.25B valuation (unicorn). ~200 employees. Reached
$100M ARR in mid-2025. Investors include Sequoia Capital
and Accel.

## Hiring

Small, high-bar engineering team (~200 people total).
Fully remote. Emphasizes software craftsmanship and
attention to detail — passion for building polished
products is the top hiring signal. Stack is intentionally
mainstream so it is not a blocker for candidates. Early
hires came from founders' networks; culture of deep work
and minimal distractions.

## Recent Context

Expanded MCP server (February 2026) with support for
initiatives, project milestones, and updates — enabling
AI coding tools to assemble context and start issues.
Rolled out advanced search filters and a UI refresh
introducing a calmer, more consistent interface. Added
Pulse, Insights, and Dashboards features. Named #7 on
Enterprise Tech 30 (2025).
