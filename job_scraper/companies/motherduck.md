# MotherDuck

## Overview

MotherDuck provides a serverless cloud analytics
platform built on DuckDB, combining local compute
with cloud-scale storage. Founded in 2022 by Jordan
Tigani (ex-Google BigQuery), Leila Horejsi, Ryan
Boyd, and Tino Tereshko, the company is headquartered
in Seattle, WA. Each user gets an isolated compute
instance ("duckling") for consistent performance
without resource contention.

## Technical Focus

- Serverless DuckDB-based cloud data warehouse
- Hybrid local + cloud query execution
- SQL analytics with zero cluster management
- Data sharing and collaboration features
- Integration with Python, dbt, and BI tools

## Scale & Stage

Private, Series B ($52.5M in Sep 2023) at a $400M
valuation, led by a16z with Madrona, Amplify,
Altimeter, Redpoint, and others. Total raised ~$100M
across 3 rounds. Approximately 50-70 employees.
Founded 2022.

## Hiring

MotherDuck hires software engineers focused on
distributed systems, query optimization, and cloud
infrastructure. The team works closely with the
DuckDB open-source community. Engineering-heavy
organization based in Seattle.

## Recent Context

MotherDuck reached general availability in June 2024.
In September 2025, the company launched a European
cloud region for serverless analytics. MotherDuck
continues to benefit from explosive DuckDB adoption,
positioning itself as the managed cloud layer for
the DuckDB ecosystem.
