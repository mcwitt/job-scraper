# dbt Labs

## Overview

dbt Labs develops dbt (data build tool), an open-source
framework that lets analysts and engineers transform
data inside cloud warehouses using SQL and Python.
Founded in 2016 by Tristan Handy, Drew Banin, and
Connor McArthur, the company is headquartered in
Philadelphia, PA. dbt has become the standard for
analytics engineering, with a large open-source
community and enterprise dbt Cloud product.

## Technical Focus

- SQL and Python-based data transformation framework
- Data modeling, testing, and documentation
- CI/CD for analytics code (dbt Cloud)
- Semantic layer for consistent metric definitions
- Integration with Snowflake, BigQuery, Databricks,
  Redshift

## Scale & Stage

Private, Series D ($222M in Feb 2022) led by Altimeter
with Databricks and Snowflake, at a $4.2B valuation.
Total raised ~$416M. Approximately 1,000 employees as
of early 2026. Founded 2016.

## Hiring

dbt Labs hires software engineers, analytics engineers,
and go-to-market roles. The engineering team builds
the open-source dbt Core framework and the managed
dbt Cloud platform. Remote-first organization.

## Recent Context

In October 2025, Fivetran and dbt Labs announced an
all-stock merger to create a combined data
infrastructure company approaching $600M in ARR.
George Fraser (Fivetran CEO) will lead the combined
entity, with Tristan Handy as co-founder and
president. The deal is expected to close within a
year, pending regulatory approval.
