# Greptile

## Overview

AI code review startup that autonomously validates code
changes in pull requests. Integrates with GitHub and
GitLab to catch bugs before they reach production.
Founded in 2023 by Daksh Gupta, Soohoon Choi, and
Vaishant Kameswaran (all Georgia Tech alumni). Y
Combinator W24 batch. Based in San Francisco.

## Technical Focus

- AI agent-based code review analyzing full codebase
  context across repos
- Integrations with GitHub, GitLab, Jira, Notion, and
  MCP servers
- Long-term memory and scoped rules for project-aware
  reviews
- Detection of .cursorrules, CLAUDE.md, and similar
  documentation for context-aware analysis

## Scale & Stage

Private, Series A. Raised $25M Series A led by
Benchmark (Sep 2025) at $180M valuation, with
participation from YC, Initialized, and Paul Graham.
~$30M total funding. ~20 employees. Reviews ~1B lines
of code per month across 7,000+ customers including
Brex, Substack, PostHog, NVIDIA, and Coinbase.

## Hiring

Small SF-based team of researchers, engineers, and
designers. All roles in San Francisco, in-person.
Hiring across engineering (research, infra, frontend,
generalist, forward-deployed), growth, and GTM. High
bar for engineering talent at a fast-moving startup.
