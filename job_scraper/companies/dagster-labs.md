# Dagster Labs

## Overview

Dagster Labs develops Dagster, an open-source data
orchestration platform built around the concept of
software-defined assets. Founded by Nick Schrock
(co-creator of GraphQL), the company is headquartered
in San Francisco, CA. Dagster provides both an
open-source project and Dagster+, an enterprise
managed platform for building data pipelines.

## Technical Focus

- Software-defined asset orchestration framework
- Declarative data pipeline definitions in Python
- Built-in data quality checks and observability
- Integration with dbt, Spark, and cloud warehouses
- Development environment with local testing support

## Scale & Stage

Private, Series B ($33M in May 2023) led by Georgian,
with 8VC, Human Capital, Sequoia, Index, and Amplify.
Total raised ~$49M. Approximately 85 employees across
5 continents. Founded 2018.

## Hiring

Dagster Labs hires software engineers, developer
advocates, and solutions engineers. The team builds
both the open-source Dagster framework and the
managed Dagster+ cloud product. Small, collegial
team with a strong open-source engineering culture.

## Recent Context

Dagster has gained significant traction as an
alternative to Apache Airflow, ranking second in
open-source orchestration adoption metrics in 2024.
The asset-centric approach differentiates Dagster
from task-based orchestrators. The company continues
to invest in AI/ML pipeline support and tighter
integration with the modern data stack.
