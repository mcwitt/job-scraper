# PromptQL (Hasura)

## Overview

AI-powered data access platform built on Hasura's Data
Delivery Network. Founded by Tanmai Gopal (CEO) and
Rajoshi Ghosh. Originally Hasura (founded 2013, GraphQL
engine). HQ in San Francisco with engineering in Bangalore.
Core engine built in Haskell.

## Technical Focus

- LLM-generated multi-step query plans via custom DSL
- Deterministic execution layer outside the LLM
- Universal data connectors (Postgres, MySQL, SQL Server)
- Intent-driven routing for query optimization
- MCP server integration for AI-data interaction
- Haskell backend (GraphQL Engine / DDN)

## Scale & Stage

Private. Hasura raised $239M total at $1B+ valuation.
PromptQL itself valued at over $1B. ~200-300 employees
across SF and Bangalore. Revenue-generating with
seven-figure enterprise deals closing in months.

## Hiring

Strong Haskell engineering culture. Roles span backend
(Haskell), data infrastructure, AI/ML, and enterprise
sales. High bar for systems programming and distributed
systems. Bangalore office handles significant portion of
core engineering.

## Recent Context

PromptQL launched as Hasura's pivot from GraphQL to AI
data access. Achieving 26-90x faster response times for
simple queries via intent routing. MCP server integration
connects to broader AI agent ecosystem. Doing seven-figure
deals while still in beta. Consulting practice launched
for enterprise AI deployment assessment.
